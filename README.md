# UIKit.plus
A collection extension for UIKit entity

## Features

- [x] CGFloat
- [x] UIWindow
- [x] UIScreen
- [x] UIView
- [x] CALayer 
- [x] UIButton
- [x] UIImage
- [x] UIImageView
- [x] UIColor
- [x] UIFont
- [x] UIDevice
- [x] UIEdgeInsets
- [x] UIScrollView
- [x] UICollectionView
- [x] UITableView
- [x] UIViewController
- [x] UINavigationController
- [x] UIAlertController
